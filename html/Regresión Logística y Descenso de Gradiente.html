Función de costes¶

La Función de Costes es una función que podemos utilizar para evaluar si nuestro algoritmo mapea la estimación objetivo, o si nuestro algoritmo realiza correctamente problemas de optimización.

En primer lugar, debemos definir la Función Sigmoide:

In [115]:
def logistic_function(x):    
    return 1/ (1 + np.exp(-x))

Definimos la Funcion de Costes:

In [116]:
def compute_cost(theta, x, y):
    m = len(y)
    y_pred = logistic_function(np.dot(x , theta))
    error = (y * np.log(y_pred)) + ((1 - y) * np.log(1 - y_pred))
    cost = -1 / m * sum(error)
    gradient = 1 / m * np.dot(x.transpose(), (y_pred - y))
    return cost[0] , gradient

Descenso del Gradiente¶

El Descenso de Gradiente es un algoritmo de optimización para encontrar un mínimo local de una función diferenciable. El Descenso de Gradiente se utiliza para encontrar los valores de los parámetros (coeficientes) de una función que minimizan una Función de Coste en la medida de lo posible.

In [118]:
def gradient_descent(x, y, theta, alpha, iterations):
    costs = []
    for i in range(iterations):
        cost, gradient = compute_cost(theta, x, y)
        theta -= (alpha * gradient)
        costs.append(cost)
    return theta, costs
